<Md>

I made Terminote as a way to take notes as fast as possible using the terminal.

- Want a note for the day? Simply write `note` in the terminal and the terminal will open Nano with your daily note.
- Name custom notes using a folder structure. If you want a note for your programming project ideas, try `note code/ideas` for example.
- If the note already exists, simply write the name of the note and it will open up for you to edit.

View the source code <Anchor href="https://github.com/dan-caldwell/terminote">here</Anchor>.
